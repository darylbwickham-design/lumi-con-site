Main controller flashing steps

This is the important bit most people will actually use. Read these steps once, then use the updater directly underneath.

  1. Connect the main controller by USB.
  2. Find the small red-and-black Firmware button on the inline switch between the Pico TX line and the ESP RX line. If it is tucked inside the case, feed it out through the gap in the casing so you can hold it.
  3. Hold the Firmware button, then click the install button in the updater below and choose the LumiCon serial device when the browser prompts you.
  4. Keep holding the Firmware button during the install.
  5. Release it once flashing is complete.
  6. If flashing fails, unplug the 6x6 keypad and try again.

Clean install option
  • Checked: performs a clean install and clears saved Wi-Fi settings and pet data.
  • Unchecked: keeps existing Wi-Fi settings and pet data where possible.
  • Use erase for recovery, a clean reset, or if the device is behaving oddly after an update.

How to flash the keypad Pico
  1. Unplug the keypad board.
  2. Hold the BOOTSEL button while connecting USB.
  3. The board should appear as a removable drive, usually named RPI-RP2.
  4. Drag the downloaded .uf2 file onto that drive.
  5. The board will copy the file, reboot, and disconnect the temporary drive automatically.
Current keypad build Latest keypad UF2 included on this site.
If the drive does not appear Unplug the board, hold BOOTSEL again, reconnect USB, and try once more.
