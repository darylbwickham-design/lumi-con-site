Before you flash

Use Chrome or Edge The browser installer for the main controller works in desktop Chrome or Edge.
Use a data cable A charge-only cable will not expose the device to the browser.
Close serial tools Close Arduino Serial Monitor and anything else using the serial port before flashing.
Know the boot buttons If flashing does not start automatically, use FLASH, BOOT, or BOOTSEL as described below.

Main controller firmware

Use this for the ESP-based main controller board.

Latest is the default recommended build and currently points to version 1.0.3.

About “Erase device”
  • Checked: performs a clean install and clears saved Wi-Fi settings and pet data.
  • Unchecked: keeps existing Wi-Fi settings and pet data where possible.
  • Use erase if you want a clean reset, are recovering a board, or are fixing odd behaviour after updating.

Keypad firmware

Use this for the keypad Pico / RP2040 board.

Download keypad UF2
How to flash the keypad Pico
  1. Unplug the keypad board.
  2. Hold the BOOTSEL button while connecting USB.
  3. The board should appear as a removable drive, usually named RPI-RP2.
  4. Drag the downloaded .uf2 file onto that drive.
  5. The board will copy the file, reboot, and disconnect the temporary drive automatically.
Current keypad build Latest keypad UF2 included on this site.
If the drive does not appear Unplug the board, hold BOOTSEL again, reconnect USB, and try once more.

Main controller flashing steps

  1. Connect the main controller by USB.
  2. Click the install button and select the LumiCon serial device when the browser prompts you.
  3. If the board does not enter flashing mode automatically, hold the FLASH or BOOT button while connecting USB or while starting the install.
  4. Release the button once flashing begins.
  5. If flashing fails, unplug the board, reconnect it, and try again.
